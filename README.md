# Example create WAR file for react app

## Installation

```bash
npm install .
// or 
yarn install
```




## 1. Installing dependencies for react app
```bash
cd src
npm install .
// or 
yarn install
```

### 2. build react app

```bash
npm run build
// or 
yarn build
```

### 3. create war file from ./src/build directory

```bash
cd ../
npm run build
// or 
yarn build

```
